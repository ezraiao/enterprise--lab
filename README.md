# enterprise--lab
Multi-zone enterprise security lab on Proxmox (Outside/DMZ/Interconnect/Inside/Secure) with dual-firewall segmentation (Palo Alto PA-440 + FortiGate 61F), GlobalProtect VPN with Duo MFA, Guacamole ZTNA, Security Onion + Splunk monitoring, and an automated SOAR pipeline (Splunk → n8n → AbuseIPDB → Palo Alto block).
